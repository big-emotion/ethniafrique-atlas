# Nom du peuple
- Nom principal : Balanta
- Auto-appellation : Balanta
- Famille linguistique : Niger-Congo – Atlantique
- Région historique : Guinée-Bissau (littoral et intérieur)
- Pays actuels : Guinée-Bissau, Gambie
- Identifiant peuple : PPL_BALANTA

# 1. Ethnies incluses
- Balanta Brassa
- Balanta Mane
- Balanta Kentohe

# 2. Origines
- Peuple agricole très ancien de la Sénégambie méridionale.

# 3. Organisation interne
- Société égalitaire (sans chefferie héréditaire).
- Décisions prises collectivement par les anciens.

# 4. Langue
- Balanta
- ISO : ble

# 5. Culture
- Rites d’initiation masculins.
- Musiques polyrythmiques.
- Cérémonies agricoles.

# 6. Histoire
- Rôle central dans l’histoire de Guinée-Bissau.
- Résistance notable au colonialisme portugais.

- Population : 620 000
- Année : 2025
