# Nom du peuple
- Nom principal du peuple : Fulɓe (Fula / Peul)
- Auto-appellation : Fulɓe (pluriel), Pullo (singulier)
- Exonymes historiques :
  - « Peul » (terme français)
  - « Fula » (forme anglophone)
  - « Fellata » (terme haoussa, parfois péjoratif → éviter)
- Pourquoi certains exonymes posent problème :
  - classification extérieure
  - « Fellata » associé à des stéréotypes historiques négatifs
- Famille linguistique principale : Niger-Congo – Atlantique (branche sénégambienne)
- Région historique : Bande sahélienne de l’Atlantique au Tchad
- Pays actuels : Sénégal, Guinée, Mali, Burkina Faso, Niger, Nigéria, Cameroun, Tchad, etc.
- Identifiant peuple (ID) : PPL_FULANI

# 1. Ethnies incluses dans le peuple
- Fulɓe Toroɓe
- Fulɓe Wodaabe
- Fulɓe Jalooɓe
- Fulɓe Kuniari
- Fulɓe Mbororo (pasteurs nomades)

# 2. Origines, migrations et formation du peuple
- Origines sénégambiennes anciennes.
- Expansions pastorales vers l’est (Sahel) entre le XVe et XVIIIe siècles.
- Rôle majeur dans les réseaux islamiques du Sahel.

# 3. Organisation et structure interne
- Structure sociale fortement pastorale.
- Classes d’âge.
- Hiérarchie lignagère.
- Autorité des chefs traditionnels (Ardo).

# 4. Langues et sous-familles
- Langue : Fulfulde / Pulaar / Pular (selon région)
- Code ISO : fuf, fuf / fub / fuh (selon pays)
- Branche : Atlantique

# 5. Culture, rites et traditions
- Pastoralisme bovin.
- Poésie élogieuse *ɗiogal*.
- Cérémonies de beauté *sharo* chez certains groupes.
- Grande tradition islamique savante (Torooɓe).

# 6. Rôle historique et interactions régionales
- États peuls : Fouta Toro, Fouta Djallon, Sokoto, Macina.
- Influence dans les réseaux sahéliens.
- Interactions historiques avec Mandinka, Hausa, Songhay.

# 7. Démographie globale
- Population : 38 600 000
- Année : 2025
