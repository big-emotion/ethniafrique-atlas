# Nom du peuple
- Nom principal : Fon
- Auto-appellation : Fɔ̀ngbè / Fon
- Appellations historiques :
  - Fon (français) → accepté
  - Dahoméens (lié à l'ancien royaume)
- Famille linguistique : Niger-Congo – Bénoué-Congo (Gbe)
- Région : Bénin (Sud), Togo
- Identifiant peuple : PPL_FON

# 1. Ethnies incluses
- Fon d’Abomey
- Fon de Porto-Novo
- Sous-groupes Mahoun, Gun, Maxi (selon zones)

# 2. Origines
- Peuple fondateur du **Royaume du Dahomey** (XVIIᵉ–XIXᵉ siècle).
- Branche Gbe issue des Adja (Tado).

# 3. Organisation
- Monarchie d’Abomey (structures historiques)
- Organisation clanique

# 4. Langue
- Fon (Fɔ̀ngbè)
- ISO : fon

# 5. Culture
- Vodun traditionnel
- Masques, divination *fa*
- Artisanat royal (bronze, bois sculpté)

# 6. Histoire
- Royaume du Dahomey, très puissant en Afrique de l’Ouest.
- Influence culturelle majeure dans le Golfe de Guinée.

# 7. Démographie
- Population : 3 500 000
- Année : 2025
