# Nom du peuple
- Nom principal du peuple : Igbo
- Auto-appellation : Ǹdị́ Ìgbò
- Exonymes / appellations historiques : « Ibo » (terme colonial réducteur)
- Famille linguistique principale : Bénoué-Congo
- Groupe ethno-linguistique : Volta-Congo → Igboïde
- Région historique : Forêt orientale du Nigeria
- Pays actuels : Nigeria, Cameroun, Guinée Éq., diaspora
- Identifiant peuple (ID) : PPL_IGBO
