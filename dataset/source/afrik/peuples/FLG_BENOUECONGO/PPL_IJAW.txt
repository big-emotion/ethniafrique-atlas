# Nom du peuple
- Nom principal : Ijaw / Ijo
- Auto-appellation : Ịjọ / Ijaw
- Famille linguistique : Niger-Congo – Bénoué-Congo (Ijoïde)
- Région : Nigeria (Delta, Bayelsa, Rivers)
- Identifiant : PPL_IJAW

# 1. Ethnies incluses
- Izon
- Okrika
- Ibani
- Andoni (éléments ijoïdes)
- Sous-groupes fluviaux

# 2. Origines
- Macro-peuple ancien du Delta du Niger.
- Société lacustre et maritime.

# 3. Organisation
- Chefferies locales
- Sociétés d’âges
- Fortes traditions fluviales

# 4. Langue
- Ijaw (plusieurs variétés)
- ISO : ijc / ijp (selon groupes)

# 5. Culture
- Masques d’eau
- Chant rituel
- Pêche et navigation deltaïques

# 6. Histoire
- Acteurs majeurs du commerce atlantique précolonial.
- Conflits pétroliers contemporains (Niger Delta).

# 7. Démographie
- Population : 14 000 000 (macro-peuple)
- Année : 2025

# 8. Sources
