# Nom du peuple
- Nom principal : Kalabari
- Auto-appellation : Kalabari
- Famille linguistique : Niger-Congo – Bénoué-Congo (Ijoïde)
- Région historique : Nigeria (Rivers State)

- Kalabari de Buguma
- Kalabari de Tombia
- Sous-groupes fluviaux

# 2. Origines
- Groupe maritime issu du macro-peuple Ijaw.
- Forte tradition liée aux rivières et lagunes.

# 3. Organisation
- Chefferies royales
- Clans fluviaux

# 4. Langue
- Kalabari (variété Ijo)
- ISO : ijn

# 5. Culture
- Masques aquatiques
- Rituels funéraires
- Navigation fluviale

# 6. Histoire
- Ancien port commercial majeur dans le delta du Niger.
- Interactions fortes avec Européens au XVIIIe-XIXe siècle.

- Population : 900 000 (2025)
