# Nom du peuple
- Nom principal du peuple : Aït Atta
- Auto-appellation : Ayt Ɛtta
- Appellations historiques : Aït Atta (forme amazighe) ; aucune connotation péjorative connue
- Famille linguistique : Afro-asiatique – Berbère (Amazigh)
- Groupe : Chleuhs / Zenata méridional selon les sources
- Région historique : Sud-est du Maroc (Saghro, Drâa, Tafilalet)
- Pays actuels : Maroc
- ID peuple : PPL_AIT_ATTA

# 1. Ethnies incluses dans le peuple
- Aït Yahya
- Aït Ouazik
- Aït Ouhman
- Divers sous-clans nomades du Saghro

# 2. Origines, migrations et formation
- Confédération amazighe structurée au cours du Moyen Âge.
- Nomadisme pastoral entre montagnes et vallées désertiques.
- Gouvernance traditionnelle fédérale.

# 3. Organisation et structure interne
- Structure clanique forte.
- Assemblées de notables (jma‘a).
- Mobilité saisonnière.

# 4. Langues et sous-familles
- Langues : Tachelhit (principalement), influences zenatiennes
- ISO : shi (pour tachelhit)

# 5. Culture, rites et traditions
- Tradition du pastoralisme ovin.
- Musiques ahwach ; danses de Saghro.
- Artisanat : cuir, tissage.

# 6. Rôle historique
- Influence dans les conflits sud-marocains.
- Résistance locale aux troupes coloniales françaises.

# 7. Démographie
- Population : 500 000
- Année : 2025

- IRCAM
