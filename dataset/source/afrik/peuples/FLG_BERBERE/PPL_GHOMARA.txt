# Nom du peuple
- Nom principal du peuple : Ghomara
- Auto-appellation : Ighomaren
- Exonymes / appellations historiques :
  - « Ghomara » (forme historique arabe et amazighe)
  - Parfois classés comme « arabisés », notion discutable (processus sociolinguistique et non identité)
- Famille linguistique principale : Afro-asiatique – Berbère (Amazigh)
- Groupe ethno-linguistique : Berbère nord-occidental (Rif occidental)
- Région historique : Nord-ouest du Rif (Maroc)
- Pays actuels : Maroc
- Identifiant peuple (ID) : PPL_GHOMARA

# 1. Ethnies incluses dans le peuple
- Tribu Aït Ghomara
- Sous-groupes villageois du Rif occidental

# 2. Origines, migrations et formation du peuple
- Population amazighe ancienne du Rif.
- Arabisation partielle depuis le Moyen Âge.
- Langue ghomari attestée comme variété berbère distincte.

# 3. Organisation et structure interne
- Structure tribale organisée en segments lignagers.
- Fort ancrage villageois.

# 4. Langues et sous-familles
- Langue : Ghomari (berbère)
- Code ISO : non attribué (langue peu documentée)
- Situation : bilinguisme arabe–amazigh fréquent.

# 5. Culture, rites et traditions
- Traditions rurales : agriculture, artisanat.
- Musiques et chants rifains.
- Rites saisonniers amazighs (Yennayer).

# 6. Rôle historique et interactions régionales
- Relations étroites avec les Rifains.
- Peu de documentation coloniale → population longtemps marginalisée dans les sources écrites.

# 7. Démographie globale
- Population totale : 100 000
- Année : 2025
