# Nom du peuple
- Nom principal : Nafusi
- Auto-appellation : Inafusen
- Appellations historiques : « Nefoussa » (forme arabe)
- Famille : Afro-asiatique – Berbère
- Région historique : Djebel Nafousa (Libye)
- Pays actuels : Libye, diaspora tunisienne
- ID : PPL_NAFUSI

# 1. Ethnies incluses
- Nafusi de Yefren
- Nafusi de Jadu

# 2. Origines
- Communautés berbères anciennes des montagnes du Maghreb.
- Résistance historique à l’arabisation complète.

# 3. Organisation
- Systèmes lignagers, assemblées locales.

# 4. Langue
- Nafusi (ISO : jbn)

# 5. Culture
- Islam ibadite (minoritaire dans le monde musulman).
- Tradition architecturale troglodyte.

# 6. Histoire
- Rôle dans les dynasties ibadites et dans l’histoire libyenne.
