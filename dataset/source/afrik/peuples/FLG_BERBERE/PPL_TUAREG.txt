# Nom du peuple
- Nom principal du peuple : Touareg
- Auto-appellation : Kel Tamasheq / Imajaghan
- Appellations historiques : « Touareg » (exonyme arabo-berbère, connotation historiquement ambiguë ; pas insultant mais marqué par les récits coloniaux)
- Famille linguistique principale : Afro-asiatique – Berbère (Amazigh)
- Groupe ethno-linguistique : Tamasheq
- Région historique : Sahara central (Ahaggar, Azawagh, Air, Adrar, Adagh)
- Pays actuels : Mali, Niger, Algérie, Libye, Burkina Faso
- Identifiant peuple (ID) : PPL_TUAREG

# 1. Ethnies incluses dans le peuple
- Kel Ahaggar
- Kel Adagh
- Iwellemmedan

# 2. Origines, migrations et formation du peuple
- Racines amazighes anciennes installées dans le Sahara.
- Formation de confédérations nomades dès l’Antiquité tardive.
- Histoire marquée par les routes transsahariennes.

# 3. Organisation et structure interne
- Confédérations (tawsit) dirigées par des amenokal.
- Sociétés hiérarchisées (nobles, tributaires, artisans).
- Nomadisme pastoral autour du dromadaire.

# 4. Langues et sous-familles
- Langue principale : Tamasheq
- Code ISO : tmh
- Sous-famille : Berbère méridional
- Usage : langue du désert saharien, écriture tifinagh.

# 5. Culture, rites et traditions
- Alfabet tifinagh ; poésie assak.
- Rituels du thé ; musiques tende et imzad.
- Place centrale des femmes dans la transmission culturelle.

# 6. Rôle historique et interactions régionales
- Grands acteurs du commerce transsaharien.
- Conflits coloniaux (France) ; rébellions modernes (Mali, Niger).
- Relations avec Songhay, Maures, Peuls.

# 7. Démographie globale
- Population totale : 4 000 000
- Année de référence : 2025
