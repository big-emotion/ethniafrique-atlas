# Nom du peuple
- Nom principal : Afar
- Auto-appellation : Qafár / Danakil
- Exonymes historiques : « Danakil » (terme européen et arabe ancien, connotation péjorative)
- Pourquoi « Danakil » pose problème :
  - utilisé par les explorateurs européens pour désigner les Afar comme groupe marginal
  - usage considéré offensant par les communautés afar
- Famille linguistique : Afro-asiatique – Couchitique
- Région : Afar (Éthiopie), Djibouti, Érythrée
- ID peuple : PPL_AFAR
