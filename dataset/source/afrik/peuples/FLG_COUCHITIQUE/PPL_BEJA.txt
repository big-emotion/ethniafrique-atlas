- Auto-appellation : Bidhaawyeet / Hadendoa
- Exonymes historiques : « Bedja » (forme européenne), « Blemmyes » (Antiquité)
- Famille linguistique : Afro-asiatique – Couchitique
- Groupe : Couchitique septentrional
- Région historique : Soudan, Érythrée, Égypte
- ID : PPL_BEJA

- Chefferies traditionnelles (nazirs).

# 4. Langue
- Beja / Bedawi
- ISO : bej

- Poésie orale.
- Pastoralisme semi-nomade.
