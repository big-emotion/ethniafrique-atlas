# Nom du peuple
- Nom principal : Gabra
- Auto-appellation : Gabra / Gabara
- Famille linguistique : Afro-asiatique – Couchitique
- Groupe : Couchitique oriental
- Région historique : Nord Kenya – Sud Éthiopie
- ID : PPL_GABRA

# 1. Ethnies incluses
- Gabra du Kenya
- Gabra d’Éthiopie (Borana zone)

# 2. Origines
- Population pastorale affiliée aux réseaux oromo et rendille.
- Ancien nomadisme à large rayon.

# 3. Organisation
- Systèmes claniques et pastoraux.
- Leadership religieux (qalicha).

# 4. Langue
- Gabra (souvent classé comme dialecte oromo)
- ISO (prox.) : orm

# 5. Culture
- Pastoralisme camelidien.
- Chant *gabbra*.
- Objets rituels en cuir.

# 6. Histoire
- Interactions fortes avec Oromo, Rendille, Somali.
- Trajectoires migratoires anciennes.

# 7. Démographie
- Population : 475 000
- Année : 2025
