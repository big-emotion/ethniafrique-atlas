# Nom du peuple
- Nom principal : Qemant
- Auto-appellation : Qemant / Qimant
- Exonymes historiques : « Falasha Qemant » (confusion coloniale avec les Beta Israel — incorrecte et problématique)
- Pourquoi l’exonyme est problématique :
  - association erronée à un autre groupe
  - classification coloniale qui nie l’identité propre des Qemant
- Famille linguistique : Afro-asiatique – Couchitique
- Groupe : Couchitique central
- Région historique : Nord-Ouest Éthiopie (Gondar)
- Pays actuels : Éthiopie
- ID : PPL_QEMANT

# 1. Ethnies incluses
- Groupes Qemant de Gondar
- Sous-groupes ruraux dispersés

# 2. Origines
- Peuple couchitique ancien ayant conservé des pratiques pré-chrétiennes.
- Parenté historique avec les Agaw.

# 3. Organisation
- Conseil religieux traditionnel : le *Wambar*.
- Communautés rurales semi-autonomes.

# 4. Langue
- Qimant (quasiment éteinte)
- ISO : ahg
- Statut : langue fortement menacée

# 5. Culture
- Rituels anciens liés aux arbres sacrés.
- Tradition agricole.
- Syncrétisme religieux avec influences orthodoxes.

# 6. Histoire
- Marginalisation progressive depuis le XIXe siècle.
- Conflits identitaires récents.

# 7. Démographie
- Population : 172 000
- Année : 2025
