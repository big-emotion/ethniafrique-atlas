# Nom du peuple
- Nom principal : Siltie
- Auto-appellation : Silte / Zayite (dans certains sous-groupes)
- Famille linguistique principale : Afro-asiatique – Couchitique
- Groupe ethno-linguistique : Couchitique oriental
- Région historique : Région Silt’e (Éthiopie)
- Pays actuels : Éthiopie
- Identifiant peuple : PPL_SILTIE

# 1. Ethnies incluses
- Silt’e du centre
- Silt’e du sud (Genale)
- Sous-groupes urbains proches de Soddo

# 2. Origines
- Population couchitique ayant subi une influence sémitique (amharique).
- Ancien sultanat local avant l’intégration dans l’Empire éthiopien.

# 3. Organisation
- Systèmes clanico-lignagers.
- Autorité religieuse influente (islam sunnite).

# 4. Langue
- Siltʼe
- ISO : stv

# 5. Culture
- Gastronomie à base d’enset.
- Musiques et chants silt’e.
- Habits et tissages traditionnels.

# 6. Histoire
- Rôle dans les royaumes musulmans du sud de l’Éthiopie.
- Identité renforcée via la création du Silt’e Zone (2001).

# 7. Démographie
- Population : 1 250 000
- Année : 2025

- Ethnologue (stv)
