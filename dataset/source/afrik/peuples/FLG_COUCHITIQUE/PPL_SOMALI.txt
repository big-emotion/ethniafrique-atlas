# Nom du peuple
- Nom principal : Somali
- Auto-appellation : Soomaali
- Exonymes historiques :
  - « Somalies » (forme coloniale française)
- Famille linguistique : Afro-asiatique – Couchitique
- Groupe ethno-linguistique : Couchitique oriental
- Région historique : Somalie historique (Somalie, Djibouti, Ogaden)
- Pays actuels : Somalie, Somaliland, Djibouti, Éthiopie, Kenya
- Identifiant peuple : PPL_SOMALI

# 1. Ethnies incluses dans le peuple
- Darod
- Hawiye
- Dir
- Isaaq
- Rahanweyn

# 2. Origines, migrations et formation
- Origines pastorales anciennes.
- Clans structurés selon la filiation patrilinéaire samal.
- Réseaux commerciaux de la Corne de l’Afrique.

# 3. Organisation
- Système clanique très structuré.
- Assemblées shir.
- Leadership par chefs traditionnels (ugaas, suldaan).

# 4. Langue
- Somali
- ISO : som

# 5. Culture
- Poésie orale (gabay).
- Élevage pastoral de chameaux.
- Rituels liés aux pluies et à la mobilité pastorale.

# 6. Histoire
- Sultanats médiévaux : Ajuran, Ifat, Adal.
- Colonisation britannique/italienne.
- Conflits contemporains.

# 7. Démographie
- Population : 27 700 000
- Année : 2025
