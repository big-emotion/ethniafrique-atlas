# Nom du peuple
- Nom principal : Macro-peuple Gur
- Auto-appellation : Diverses selon les sous-groupes
- Appellations historiques : « Voltaïque » (terme ancien, parfois maladroit car trop large)
- Pourquoi « Voltaïque » pose problème :
  - terme administratif colonial utilisé pour regrouper des populations très diverses
  - non pertinent scientifiquement comme catégorie ethnolinguistique
- Famille linguistique : Niger-Congo – Gur
- Région : Burkina Faso, Ghana, Togo, Bénin, Côte d’Ivoire
- Identifiant : PPL_GUR_MACRO

# 1. Ethnies incluses
- Mossi, Dagomba, Mamprusi, Kassena, Dagara, Lobi, Frafra, Gurma, Konkomba, Nankana, Kusasi, Bwa, Bissa, Wiyã, Talensi, etc.

# 2. Origines
- Grand ensemble voltaïque ancien.
- Heterogénéité culturelle mais cohérence linguistique.

# 3. Organisation
- Pas de structure unique.
- Présence de royaumes (Mossi, Dagbon) et de sociétés segmentaires (Lobi, Dagara).

# 4. Langues
- Ensemble des langues Gur (Mooré, Dagbani, Kasem, Dagaare, Kusaal, etc.)

# 5. Culture
- Masques, rituels agraires, bois sacrés.
- Architecture en terre.

# 6. Histoire
- Origine des états mossi.
- Réseaux culturels du Volta supérieur.

# 7. Démographie
- Population totale : 0 (catégorie macro-groupe)
- Année : 2025
