# Nom du peuple
- Nom principal : Sénoufo
- Auto-appellation : Senufo / Syénfo
- Appellations historiques : Sénoufo (forme française), Syenfo
- Famille linguistique : Niger-Congo – Gur (élément Gur dans le cadre AFRIK)
- Région historique : Côte d’Ivoire, Mali, Burkina Faso
- Identifiant peuple : PPL_SENUFO

# 1. Ethnies incluses
- Sénoufo Tagwana
- Sénoufo Nyarafolo
- Sénoufo Kafibele
- Sénoufo Nafara

# 2. Origines
- Civilisation ancienne du nord ivoirien.
- Liens culturels avec Lobi, Koulango, Mandé.

# 3. Organisation
- Société initiatique du *Poro* (structure centrale).
- Lignages patrilinéaires.

# 4. Langue
- Sénoufo (plusieurs variétés)
- ISO : sef / ses / qaz (selon zones)

# 5. Culture
- Sculptures sénoufo célèbres (dans l’art africain classique).
- Masques *kponyugo*.
- Rituel du Poro (initiation masculine).

# 6. Histoire
- Relations anciennes avec Kong, Dyula et peuples voltaïques.
- Structuration rituelle très complexe.

# 7. Démographie
- Population : 2 700 000
- Année : 2025

# 8. Sources
- Ethnologue
- Côte d’Ivoire Northern Cultural Archives
- Musée des Civilisations Sénoufo
