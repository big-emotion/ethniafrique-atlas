# Nom du peuple
- Nom principal : Guéré / Wè
- Auto-appellation : Wè
- Exonymes historiques :
  - Guéré (terme ivoirien français, accepté mais extérieur)
- Famille linguistique : Niger-Congo – Krou
- Région historique : Ouest ivoirien (Duékoué, Guiglo), Liberia
- Identifiant : PPL_GUERE

# 1. Ethnies incluses
- Wè du nord
- Wè du sud

# 2. Origines
- Peuple forestier ancien du massif du Cavally.
- Proches linguistiquement des Yacouba (Dan) mais Gur dans classification AFRIK.

# 3. Organisation
- Structure lignagère
- Sociétés initiatiques
- Autorité du « maître de la terre »

# 4. Langue
- Wè (northern : wec ; southern : wew)
- ISO : wec / wew

# 5. Culture
- Masques *glé*
- Rituels du bois sacré
- Chants polyphoniques

# 6. Histoire
- Zones de résistance durant la colonisation
- Conflits internes dans les années 2000 en Côte d’Ivoire

# 7. Démographie
- Population : 820 000
- Année : 2025

# 8. Sources
- Ethnologue
- Archives culturelles de l’Ouest ivoirien
