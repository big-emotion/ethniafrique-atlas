# Nom du peuple
- Nom principal : Soninké
- Auto-appellation : Soninke / Sooninke
- Exonymes historiques : Sarakolé (terme utilisé mais perçu comme extérieur ; non péjoratif mais moins préféré)
- Famille linguistique : Niger-Congo – Mandé
- Région historique : Mauritanie, Mali, Sénégal
- Identifiant : PPL_SONINKE

# 1. Ethnies incluses
- Soninké du Gajaaga
- Soninké du Waalo
- Soninké du Guidimakha

# 2. Origines
- Peuple fondateur de l’Empire du Ghana (VIIIe–XIe siècle).
- Un des groupes mandé les plus anciens.

# 3. Organisation
- Systèmes de castes (forgerons, griots, etc.).
- Chefferies lignagères.

# 4. Langue
- Soninke
- ISO : snk

# 5. Culture
- Rites anciens.
- Musiques mandingues.
- Riches traditions orales.

# 6. Histoire
- Empire du Ghana.
- Diaspora importante dans toute l’Afrique de l’Ouest.

# 7. Démographie
- Population : 2 500 000
- Année : 2025
