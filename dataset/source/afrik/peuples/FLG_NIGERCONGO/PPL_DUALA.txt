# Nom du peuple
- Nom principal du peuple : Duala
- Auto-appellation : Duàlá
- Exonymes / appellations historiques : Douala (orthographe francisée) ; « Jalla » (ancienne graphie étrangère)
- Famille linguistique principale : Niger-Congo (FLG_NIGERCONGO)
- Groupe ethno-linguistique : Bantous côtiers (Sawa)
- Région historique : Littoral camerounais (Wouri)
- Pays actuels : Cameroun
- Identifiant peuple (ID) : PPL_DUALA

# 1. Ethnies incluses dans le peuple
- Ethnie 1 : Duala de Bonanjo
- Ethnie 2 : Duala de Bonaberi
- Ethnie 3 : Sous-groupes Sawa apparentés (éléments Bakoko, Isubu… selon contextes)

# 2. Origines, migrations et formation du peuple
- Origines anciennes : Installation sur les bords de l'estuaire du Wouri.
- Routes migratoires : 
  - Mobilité entre littoral et zones maritimes
  - Expansion vers les zones urbaines
- Unifications / divisions : Formation progressive de lignages marchands dominants.
- Influences externes : Contacts très anciens avec Européens (XVIᵉ siècle), rôle majeur dans le commerce atlantique.

# 3. Organisation et structure interne
- Système politique traditionnel : Chefferies notables (Bell, Akwa, Deido…).
- Organisation clanique : Clans lignagers consolidés autour de familles nobles.
- Rôle des lignages : Alliances commerciales, gestion côtière.
- Autorité religieuse : Mélange de cultes ancestraux et christianismes anciens.

# 4. Langues et sous-familles
- Langue principale : Duala
- Codes ISO : dua (ISO 639-3)
- Dialectes : 
  - Variantes urbaines et rurales
- Rôle véhiculaire : Langue historique du commerce côtier camerounais.

# 5. Culture, rites et traditions
- Rites majeurs : 
  - Cérémonies maritimes
  - Fêtes sawa
  - Rites funéraires élaborés
- Symboles : Estuaire du Wouri, pirogues, totems lignagers
- Arts et musique : 
  - Chant
  - Danse sawa
  - Sculptures maritimes
- Spiritualités : Croyances en ancêtres, forte présence du christianisme.

# 6. Rôle historique et interactions régionales
- Royaumes / chefferies : Familles Bell & Akwa dominantes.
- Relations avec peuples voisins : Intenses échanges avec Bakweri, Bassa, Bakoko, Mabea…
- Conflits / alliances : Alliances commerciales précoloniales ; résistance anti-allemande (Ngondo).
- Diaspora : Présence à Douala, Yaoundé et à l'étranger.
