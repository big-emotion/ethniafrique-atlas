# Nom du peuple
- Nom principal : Kafa
- Auto-appellation : Kafficho / Kaafi
- Exonymes historiques : « Kaffa » (forme européenne ancienne) ; pas péjoratif mais extérieur
- Famille linguistique : Afro-asiatique – Omotique
- Groupe : Omotique sud
- Région historique : Kaffa Zone (Éthiopie), hauteurs du sud-ouest
- Pays actuels : Éthiopie
- Identifiant peuple : PPL_KAFA

# 1. Ethnies incluses
- Kafa des hautes terres
- Groupes ruraux forestiers

# 2. Origines
- Région considérée comme berceau du café (Kaffa → coffee).
- Société omotique ancienne dotée d’un royaume historique.

# 3. Organisation
- Monarchie traditionnelle passée (*Geba*).
- Systèmes lignagers.

# 4. Langue
- Kafa / Kafficho
- Code ISO : kbr

# 5. Culture
- Culture du café (patrimoine historique).
- Artisanat : travail du bois, décorations textiles.

# 6. Histoire
- Royaume de Kafa jusqu’à la conquête par Menelik II (XIXe siècle).
- Relations avec le royaume du Jimma et les Wolayta.
