# Nom du peuple
- Nom principal : Mafa
- Auto-appellation : Mafahay
- Exonymes historiques : aucun terme péjoratif important identifié
- Famille linguistique : Afro-asiatique – Tchadique
- Région historique : Montagnes du Mandara (Extrême-Nord Cameroun)
- Pays actuels : Cameroun, Nigeria (minorité)
- Identifiant peuple : PPL_MAFA

# 1. Ethnies incluses
- Mafa des Monts Mandara
- Variantes locales villageoises

# 2. Origines
- Population montagnarde ayant développé des systèmes agricoles complexes.
- Habitat en terrasses et architecture defensive.

# 3. Organisation
- Systèmes lignagers.
- Chefferies locales (village headmen).

# 4. Langue
- Mafahay (ISO : maf)

# 5. Culture
- Masques, rites de montagne.
- Tissage, poterie, métallurgie traditionnelle.

# 6. Histoire
- Populations défensives face aux anciens raids esclavagistes.
- Relations avec les Mandara, Kapsiki, Giziga.
